## 🔴 P0 - API 規格與認證

### Token 簽名不匹配（認證失效）
| 問題 | 嚴重程度 | 發現者 |
|------|----------|--------|
| 前端 Token 簽名是硬編碼 `dev_signature`，與 Server HMAC-SHA256 驗證完全不匹配 | 🔴 P0 | 光一、深、光三 |
| 前端 `generateToken()` 使用靜態字串，Server 端動態計算 | 🔴 P0 | 光一 |

### 安全密鑰問題
| 問題 | 嚴重程度 | 發現者 |
|------|----------|--------|
| AUTH_SECRET 硬編碼 `"gapi_dev_secret_key_change_in_production"` | 🔴 | 光一、光二、光三 |
| 開發模式繞過認證 - `verify_auth` 返回 `None` 允許訪問 | 🔴 | 光三、深 |
| background.js 使用固定 `dev_signature` 繞過 HMAC 驗證 | 🔴 | 深 |

### Session 與 CORS 問題
| 問題 | 嚴重程度 | 發現者 |
|------|----------|--------|
| Session 清理機制缺失 | 🟠 | 光一 |
| CORS 設定過寬 (`origins = ["*"]`) | 🟡 | 光一、光三 |
| Token 解析底線問題 - extension_id 包含底線時解析錯誤 | 🟡 | 光三 |
