## 技術筆記

### 常見問題與解決方案
| 問題 | 原因 | 解決方案 |
|------|------|---------|
| 找不到輸入框 | Gemini 使用 Shadow DOM 或動態載入 | 使用 waitForSelector 等待 |
| 輸入無效 | 事件未觸發 | 使用 dispatchEvent 觸發 input 事件 |
| 截圖失敗 | 瀏覽器已關閉 | 使用 try-catch 包裹 |
| 網路超時 | 頁面載入慢 | 使用 waitUntil: 'networkidle' |

### 選擇器策略
1. **ARIA**: `button[aria-label="Send"]`
2. **Placeholder**: `textarea[placeholder*="message"]`
3. **Contenteditable**: `div[contenteditable="true"]`
4. **Text**: `text=Send`
5. **JavaScript**: `page.evaluate()` 直接操作 DOM
